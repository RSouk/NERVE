Email Configuration

Email Configuration Setup

Not Configured

What Email is Used For:

  • Password reset emails ("Forgot password" flow)
  • Welcome emails when creating new users
  • Security alerts (failed logins, account locked)
  • Scan completion notifications (optional)
  • Account deletion confirmations

Setup Steps (Before Production):

  1. Buy Domain: nerveplatform.com (or your preferred domain)
  2. Choose Email Provider:
    • SendGrid (recommended) - 100 emails/day free tier
    • Mailgun - Good for high volume
    • AWS SES - Cheapest for production scale
    • Gmail SMTP - Only for testing (not production)
  3. Setup Email Address: noreply@nerveplatform.com or alerts@nerveplatform.com
  4. Get SMTP Credentials: Provider will give you:
    • SMTP Host (e.g., smtp.sendgrid.net)
    • SMTP Port (usually 587 for TLS)
    • Username (API key or username)
    • Password (API secret)
  5. Configure DNS: Add SPF, DKIM, DMARC records for deliverability
  6. Enter credentials below and test
For Now: Platform works without email configured. Password resets and notifications will be disabled until email is set up. Configure before production launch.
